## Introduction
> 自己制定的功能比较全面的vue.js框架
- 支持stylus及各种css样式的import，.vue中引入css要用@import，js中require，import都可以
- 支持vuex
- 支持vue-router
- 支持vue-resource
- 支持element-ui
- 支持exprss服务器搭建
